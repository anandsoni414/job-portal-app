# Job Portal Frontend

React + Vite + Tailwind frontend for the Job Portal app.

## Run

```bash
npm install
copy .env.example .env
npm run dev
```

## Environment

```env
VITE_API_URL=http://localhost:5000/api
```

## Pages

- `/` - job listing and search
- `/jobs/:id` - job details and apply
- `/login` - login
- `/register` - register as student or recruiter
- `/profile` - profile and resume URL
- `/applications` - student applications
- `/recruiter` - recruiter dashboard
